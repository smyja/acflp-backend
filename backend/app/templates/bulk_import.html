JSONL Format Example:
{"title": "Translate greeting", "source_language": "english", "target_language": "bini", "content": "Hello, how are you?", "description": "Simple greeting translation", "task_type": "text_translation", "reward_amount": 5.00}
{"title": "Translate farewell", "source_language": "english", "target_language": "urhobo", "content": "Goodbye, see you later", "description": "Farewell translation", "task_type": "text_translation", "reward_amount": 4.50}
Required Fields:
  • title: Task title (max 255 characters)
  • source_language: Source language for translation
  • content: Text content to be translated (max 5000 characters)
Optional Fields:
  • description: Task description (max 1000 characters)
  • task_type: Type of task (default: "text_translation")
  • target_language: Target language for translation
  • reward_amount: Reward amount in decimal format (default: 0.00)
